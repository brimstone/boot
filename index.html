brimstone's boot iso/kernel

.iso - .kernel

This is a custom build of ipxe that calls back to a script at this url when it boots. This way, the installer files for various Linux operating systems are downloaded at every boot, directly from the appropriate mirrors. This keeps the total download for you small and fast, as well as keeping the installer files up to date.

Provided are two download options:

iso format - This format is a valid iso9660 CD image, ready to be used in a VM, burned to optical media, you name it. This file can be directly written to a USB drive as well.

kernel format - This format is a valid ELF kernel. It can be used where you already have a boot environment, perhaps an existing PXE setup, a USB drive with grub4dos or syslinux.
